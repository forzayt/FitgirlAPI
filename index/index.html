Game Data Aggregator

A robust, high-performance REST API that acts as a bridge, dynamically merging rich official Steam metadata with powerful, backed download links from the official FitGirl Repacks website into a cohesive JSON experience.

BASE URL
https://fitgirlapi.onrender.com

Available Endpoints

Seamlessly integrate game data into your frontend application with these intuitive routes.

GET /api/v1/games

List All Games

Asynchronously scans the data repository and returns a comprehensive array of all available games based on stored configurations.

Response Data
games Array of Game IDs stored in the directory
GET /api/v1/popular

Get Popular Games

Fetches the top 100 most popular games from SteamSpy and filters them against your local database, ensuring only fully archived popular titles are returned.

Response Data
popular Array of popular Game IDs mapped with local data
GET /api/v1/:id

Get Game Details

The core aggregator endpoint. Concurrently queries the Steam API for rich metadata while parsing JSON/Text files containing backed links from the official FitGirl website.

Parameters & Data
:id The official Steam App ID (e.g., 271590 for GTA V)
response Combined payload of steam_data and download_data
